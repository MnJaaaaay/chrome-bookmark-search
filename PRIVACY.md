# 隐私政策 / Privacy Policy

**适用范围**：Chrome 扩展「书签搜索助手」（以下简称"本扩展"）
**最近更新**：2026-04-29

## 一、我们收集的数据

本扩展会在你的本地浏览器中读取或存储以下数据：

| 数据类型 | 来源 | 用途 | 存储位置 |
|---------|------|------|---------|
| 书签 | `chrome.bookmarks` API | 在弹窗中搜索与展示 | 仅在内存中读取，不落盘 |
| 浏览历史 | `chrome.history` API | 在弹窗中搜索与展示 | 仅在内存中读取，不落盘 |
| 剪贴板复制内容 | 监听网页 `copy` 事件中所选文本 | 提供"剪贴板历史"快速回粘功能 | `chrome.storage.local`（仅本机） |
| 用户自建的"常用片段" | 用户在弹窗中手动新增 | 一键复制常用文本 | `chrome.storage.local`（仅本机） |

## 二、我们不做的事情

- **不会**把任何数据上传到任何服务器；本扩展没有任何远程网络请求。
- **不会**把数据共享、出售或交换给第三方。
- **不会**收集你的身份信息、账号、Cookie、密码或表单数据。
- **不会**对页面 DOM 做任何改写，content script 只监听 `copy` 事件。

## 三、剪贴板数据说明（重要）

剪贴板历史功能通过监听网页中的 `copy` 事件，把你**主动选中并复制**的文本保存到本地。

- 默认最多保留最近 100 条；超出后自动删除最旧的一条。
- 你可以随时在扩展弹窗的「剪贴板」标签下点击「清空历史」一键删除所有记录。
- 卸载本扩展时，Chrome 会一并删除 `chrome.storage.local` 中的所有数据。

如果你不想被记录，请在复制敏感内容（密码、银行卡号等）前先在 `chrome://extensions/` 中临时禁用本扩展。

## 四、所申请的权限及理由

| 权限 | 用途 |
|------|------|
| `bookmarks` | 读取书签树用于搜索 |
| `history` | 读取浏览历史用于搜索 |
| `tabs` | 点击搜索结果时新建标签页打开网址 |
| `storage` | 在本机保存剪贴板历史与常用片段 |
| `<all_urls>` content script | 在任意网页监听 `copy` 事件以采集复制文本 |

## 五、联系方式

如对本隐私政策有任何疑问，请通过 Chrome 应用商店上架页面的开发者邮箱联系我们。
